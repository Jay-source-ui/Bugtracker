Bug Tracker REST API

Base URL: http://localhost:8082/api

Bug Endpoints

GET /api/bugs

Get all bugs

GET /api/bugs/{id}

Get bug by ID

POST /api/bugs

Create new bug

Body: {"title": "Bug title", "description": "Description", "priority": "HIGH"}
PUT /api/bugs/{id}

Update bug

DELETE /api/bugs/{id}

Delete bug

PATCH /api/bugs/{id}/resolve

Resolve bug

GET /api/bugs/status/{status}

Get bugs by status (OPEN, IN_PROGRESS, RESOLVED)

GET /api/bugs/priority/{priority}

Get bugs by priority (HIGH, MEDIUM, LOW)

GET /api/bugs/search?keyword={text}

Search bugs by title

Statistics Endpoints

GET /api/stats/dashboard

Get dashboard statistics

User Endpoints

GET /api/users

Get all users

GET /api/users/{username}

Get user by username

GET /api/users/role/developer

Get all developers

GET /api/users/role/tester

Get all testers

Test API with cURL

Get all bugs:
curl -X GET http://localhost:8082/api/bugs
Create a bug:
curl -X POST http://localhost:8082/api/bugs \
  -H "Content-Type: application/json" \
  -d '{"title":"API Test","description":"Test bug","priority":"HIGH"}'
Resolve a bug:
curl -X PATCH http://localhost:8082/api/bugs/1/resolve
Delete a bug:
curl -X DELETE http://localhost:8082/api/bugs/1
